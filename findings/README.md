# findings

Interpreted, evidence-backed observations with explicit lifecycle states (`observed`, `stale`, or `superseded`). Generated by `procedures/boundary-drift-recon-v0.md`.

**Schema:** `schemas/finding.schema.json`

**MVP lifecycle (§9):** Only `observed`/`stale`/`superseded`. No `acknowledged`, `disputed`, `accepted_deviation`, or `resolved`.

**Navigation:** Finding → `evidence_refs` → Evidence → Pin.
